# Project Aplikasi Manajemen Data Mahasiswa
## Deskripsi Aplikasi

Aplikasi Manajemen Data Mahasiswa adalah aplikasi berbasis web yang dibuat untuk memenuhi tugas **Ujian Akhir Semester (UAS) Mata Kuliah Pemrograman Web**.  
Aplikasi ini digunakan untuk mengelola data mahasiswa yang meliputi NIM, nama, jurusan, dan alamat.

Aplikasi dibangun menggunakan bahasa pemrograman **PHP**, database **MySQL**, serta menerapkan konsep **MVC (Model View Controller)**.  
Tampilan antarmuka dibuat menggunakan **Bootstrap 5** agar responsif dan mudah digunakan.

## Fitur Aplikasi
### 1. Login dan Role User
Aplikasi memiliki sistem login dengan dua role pengguna:
- **Admin**
  - Menambah data mahasiswa
  - Mengedit data mahasiswa
  - Menghapus data mahasiswa
  - Melihat data mahasiswa
- **User**
  - Login ke sistem
  - Melihat data mahasiswa (read only)

---

### 2. Manajemen Data Mahasiswa (CRUD)
- Tambah data mahasiswa
- Tampil data mahasiswa
- Edit data mahasiswa
- Hapus data mahasiswa

---

### 3. Pencarian Data (Search)
- Pencarian berdasarkan:
  - NIM
  - Nama Mahasiswa

---

### 4. Pagination
- Data ditampilkan per halaman
- Setiap halaman menampilkan 5 data
- Tersedia tombol Previous, Next, dan nomor halaman

---

### 5. Tampilan Antarmuka
- Menggunakan Bootstrap 5
- Tampilan responsif
- Background full halaman
- Header dengan desain gradient
- Menggunakan Card, Table, dan Button Bootstrap

---

## Teknologi yang Digunakan
- PHP
- MySQL
- Apache (XAMPP)
- Bootstrap 5
- HTML & CSS
- Konsep MVC (Model View Controller)
